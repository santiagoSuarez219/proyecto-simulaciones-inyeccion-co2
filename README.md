# Modelo-ITM

Modelo de deep learning (FNO + FiLM) para predicción espacio-temporal del Factor de
Seguridad (SF) y la Deformación Volumétrica (VD) en reservorios depletados bajo
inyección de CO₂. Ver `CLAUDE.md` para la documentación completa del proyecto,
estructura de carpetas, arquitectura y convenciones.

Estado: estructura de proyecto en migración. El código de entrenamiento vive
actualmente en `../Codigo Entrenamiento/train_dataset.py`; su migración a
`src/modelo_itm/` está pendiente (ver `specs/`).
